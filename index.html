I Need Better Publishing Tools

Dec 3rd, 2012

Publishing with Google Drive as content sharing repository.

We had situation where we can share the content before, like Drop Box, WebHard, iCloud. So, why Google Drive is special?

* It supports multi-platforms unlike iCloud, which is mostly geared for Apple centric workflow.

* It has the content editing capability unlike DropBox and several other services, which has mostly storage capabilities.

* It has API so that developers can use it with their programs. I can get information and edit the same data from the spreadsheet of customers using my program, if I know the user id and the password.

This opens up a whole new way of publishing.

I can imagine a publishing serve platform like the following.

Case Study

1. Catalog Reseller

I am a reseller of pharmaceutical products. I sell a several thousands products. I need to make catalogs all the time for various catalog and various distribution channel. But it not so easy.

  1. It is very time consuming

  2. It is very expensive to create a paper catalog or web site.

And I would like some serve that can handle this very efficiently, and I am willing to pay for it.
